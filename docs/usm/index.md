# User Story Mapping (USM) - CellVex

## O que é User Story Mapping?

O User Story Mapping (USM) é uma técnica visual e colaborativa para o desenvolvimento de software, criada por Jeff Patton. Seu objetivo é organizar e priorizar histórias de usuário de forma estruturada, facilitando a visualização da jornada do usuário e a definição de prioridades do produto.

## Benefícios do USM

- Organiza atividades principais em uma linha do tempo.
- Distribui histórias menores e tarefas por prioridade.
- Facilita a identificação das funcionalidades mais importantes.
- Promove colaboração entre equipes e stakeholders.
- Ajuda a identificar lacunas, redundâncias e fluxos desnecessários.
- Alinha o backlog à estratégia do projeto.
- Adapta-se bem a metodologias ágeis.

## Aplicação no Projeto CellVex

No projeto CellVex, o USM foi utilizado para mapear a experiência do usuário e definir as funcionalidades essenciais do sistema, garantindo entregas alinhadas às necessidades do cliente.
